    Method of Use

    Combination Keys’ function

    Press FN+Left Ctrl to switch to the second function of Combination Keys(except Arrow Keys).Press FN+Ctrl again,to restore the 9 keys main function Y/U/I/H/J/K/N/M/<,but the other Combination Keys still in the second function.Press FN+Ctrl third times,all the Combination Keys’ main function recovery.You can also lock/unlock Arrow Keys at any time by pressing FN+Enter

    Use the Shortcut Keys

    FN+A/S=Switch Win/Mac

    FN+(F1-F12)=Achieve functions such as: Previous track,Mute and so on(Only for MacOs)
