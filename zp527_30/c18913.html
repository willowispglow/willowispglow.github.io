■ AESとは
AES樹脂とは、A ( アクリロニトリル ) + E ( エチレン ) + S ( スチレン ) から構成されるプラスチック素材。
ABS樹脂の B ( ブタジエン ) の代わりに、E ( エチレン ) を使用しており、その特性は耐候性に優れるため屋外での使用に適し、耐傷付き性にも高い効果を発揮します。
また、エムズスピードで使用するAESは、その特有の素材感を活かし、塗装せずともそのまま使用出来るメリットがあります。

■ ABSとは
ABS樹脂とは、A ( アクリロニトリル ) + B ( ブタジエン ) + S ( スチレン ) から構成されるプラスチック素材。
耐久性に加え、性質上の特性から硬度があるため、外部からの力に対する耐衝撃性に優れています。
こちらは、塗装して使用する製品となります。
