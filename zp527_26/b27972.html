Features

  • Comply with IEEE802.3、IEEE802.3i、IEEE 802.3u、IEEE802.3ab、IEEE802.3x、IEEE802.3az、IEEE802.1q Ethernet standards;
  • Supports IEEE802.3af、IEEE802.3at PoE standards;

  • Support 5 x10/100/1000Mbps Auto-negotiation RJ45 ports with 4 port PoE function  (port1-port4);

  • LED indicators for monitoring Port Link / Activity;
  • External power supply, exquisite design, suitable for desktop 、Wall hanging install and use.
