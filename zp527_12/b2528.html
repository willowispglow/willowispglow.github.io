ポリレジンの文字盤のレイヤーやウッドの質感がデザイナブル

木目が美しいウッドフレームに、ポリレジンの文字盤をレイヤードさせたナチュラルなウォールクロックです。角度が変わると、くり抜いた数字部分から後ろに重ねたウッドフレームが覗くようにデザインしています。大きな数字や高い視認性など、実用性にもこだわりました。

- Features -
Function 01
レイヤー

角度が変わると、くりぬいた数字部分から後ろに重ねたウッドフレームが覗く奥行きのあるデザインです。淡いトーンの濃淡が味わえるメープル材を使用しています。
Function 02
大きめ

数字は文字盤に対して少し大きめに。視認性とデザインを考慮し連なるようにバランスよく配置しました。
Function 03
奥行き

文字盤はウッドフレームよりも少し厚みを持たせて奥行きを出しています。角を落とすことでフレームとの調和を図っています。ポリレジンとの異素材感が際立ちます。
Function 04
音がしない

針が進む際の音がしない静音タイプのムーブメントを使用しているので、寝室など静かな場所でも気兼ねなくお使いいただけます。
Function 05
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

- Brand information -

「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、
みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
