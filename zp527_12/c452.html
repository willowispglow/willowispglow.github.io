商品コメント 1. The FS-S6X1-Pro is a PCI express 3.0 X1 interface to 6-Ports 6 Gbps max speed SATA III expansion card. It allows users add 6X NONE-RAID SATA III storage or AHCI BOOT UP HDDs/SSDs on Desktop PCs, Working Stations, MAC Pros and Network Attached Storage (NAS).
2. Select the ASMedia ASM1166 the most stable NONE-RAID PCIE 3.0 SATA III host controller. PCIE 3.0 X1 interface provide 8Gbps total bandwidth. Each SATA III port will get 6Gbps max speed for AHCI BOOT UP or storage.
3. Plug and Play on Windows 11, 10, 8.x, 7, Vista, XP (32/64bit) and Windows Server 2022, 2019, 2016, 2012R2, 2012, 2008R2, 2008, 2003R2, 2003 (32/64). MAC OS 10.4.x, 10.5.x, 10.6.x, 10.7.x, 10.8.X, 10.9.x, 10.10.x, 10.11.x, 10.12.x, 10.13.x, 10.14.x, 10.15.x and most of Linux kernel systems.
4.Compatible with Serial ATA(SATA) Revision 3.1 specifications, auto switch to SATA 3.0 6Gbps max speed. Backward compatible with SATA 1.0 1.5Gbps speed and SATA 2.0 3Gbps speed. Support PCIE 1.0, 2.0, 3.0, 4.0 protocol, works on motherbaord PCIE X1, X2, X4, X8, X16 slot.
5.WARNING: 1. This SATA card was set on AHCI BOOT UP mode, make sure now PCs SSDs/HDDs in AHCI BOOT UP mode or will report Blue Screen Death (BSD). 2. Make sure PCs power supply unit (PSU) can provide 6X max SATA interface power connector. 3. Slim PCs need user change to low profile bracket.
