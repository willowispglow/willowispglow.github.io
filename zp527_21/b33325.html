ST-01 PRO は、ST-01のすべての機能を保持することに加えてるアップグレードバージョンです。
1. ST-01 PRO は USBメモリー入力をサポートし、ロスレス形式の音楽を再生できます。
2. 底面に VU メーターの感度調整を追加し、設定が簡単で、さまざまな入力信号との適合性を向上させます。
3.真空管とVUメーターの周りに3つの装飾的な金色のリングを追加し、より精巧で豪華に見えます。
4. PC-USB 入力は OTG 機能を備えたスマホにも対応していますが、Android システムには OTG ケーブル、IOS システムにはカメラキットが必要です。ご自身でご用意ください。
5. プラグ可能な真空管ソケット、デフォルトの 6K4 真空管は、6J1 / 6J2 / 6*1n / 5654 / 6J4 / 6J5 / 6K5 / 6A2 などに交換できます。

ユニークで驚くべき組み合わせ: ST-01/ST-01 PRO アンプと VU メーターが一体となった一体型のコンパクトなサイズで、HiFi 音楽を楽しむとヴィンテージ感のある温かみのある光の中でポインターが舞い、オーディオ空間をダイナミックに飾ります。
トーンコントロールを備えたHiFiクリアサウンド：共鳴回路と簡潔な配線、真空管を採用して音を柔らかくし、デジタル感を取り除き、ノイズのない暖かく甘い音を出します。また、自分の好みのサウンド スタイルを調整するための高音と低音のノブを使用して、HiFi ミュージックに刺激的な高音と深い低音を提供します。
ハイパワーと特別なAUX出力：TI社のクラシックパワーアンプICを使用し、各チャンネルは100W（4Ω）を出力でき、すべての3-8Ωパッシブスピーカーを簡単に駆動できます。3.5mm AUX出力を追加して、別のアンプまたはアクティブスピーカー/サブウーファーを簡単に接続して、マルチチャンネルシステムを構築し、素晴らしいホームシアター体験がお楽しめます。
複数のデジタルオーディオ入力: Bluetooth 5.0 / USB / 同軸 / 光 / ステレオ RCAで、スマホ、コンピューター、タブレット、テレビ、テレビ、CD プレーヤーなどのほとんどのオーディオ デバイスと接続できます。DAC とアンプの完璧な組み合わせで、さまざまなオーディオ機器を所有するオーディオファンにおすすめです。
使いやすさを追求した実用的な機能: フル機能のリモコンを使用すると、離れた場所からすべての設定を行うことができます。入力の選択とインジケーターが搭載され、作業状況をリアルタイムで明確に表示します。アルミニウム合金ケースは特別にデザインされ、クールでユニークでデスクトップスペースを飾ります。

ご注意：

1.本機の感度は、99％のユーザーのニーズを満たすように工場で設定されているため、本機を入手した後に調整する必要はありません。
2.たまたま1％のユーザーであった場合、抵抗器は小さく、精密機器であるため、時計回りまたは反時計回りにゆっくりと調整してください。
3.抵抗器の調節の限界は 210° であり、 360°調節ではありません。従って調節可能な範囲は終点があります、あなたが終点に調節するとき決して再度調節しないで下さい。
4.一定期間普通に使っていって、急にVUメーターが振れなくなったら、左右にそっと調整すると、回復可能と思います。この状況はちょっとした接触不良が原因かもしれません。
5.曲を切り替えたり、入力チャンネルを切り替えてもVUメーターが振れない場合は、入力される音楽信号が小さすぎることが主な原因です。
